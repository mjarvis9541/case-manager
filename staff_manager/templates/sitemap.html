Case Types [Complete]

Views to CREATE, RETREIVE, UPDATE and DELETE (CRUD) for creating new case types and setting their case value. (For example ID Proofs are currently worth 15 mins - you can change the minute value for the cases here - you can also create new case types such as 'PPI/No PPI' etc...)

Case type list view - view a list of case types

Within this view:

  • Case Type create - create a new case type (PPI, No PPI, IDV Fail, etc.) and assign minute value
  • Case Type detail - view the details of a case type type
  • Case Type update - update a case type (name, minute value
  • Case Type delete - delete a case type
